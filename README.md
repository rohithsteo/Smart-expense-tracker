# 💰 Smart Expense Predictor + Insights

A Python-based system that tracks expenses, analyzes spending patterns, and predicts future expenses using machine learning.

---

## 🧠 Overview

This project allows users to record expenses, analyze spending habits, and predict future expenses using a Linear Regression model.

It also detects potential overspending trends.

---

## ⚙️ Features

- Add daily expenses
- Category-wise spending analysis
- Total expense calculation
- Future expense prediction
- Overspending alerts

---

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- SQLite

---

## 📂 Project Structure

main.py  
requirements.txt  
README.md  

---

## 🚀 How to Run

1. Install dependencies  
pip install -r requirements.txt  

2. Run the program  
python main.py  

---

## 🧠 How It Works

1. Stores expenses in SQLite database  
2. Uses Pandas for analysis  
3. Applies Linear Regression for prediction  
4. Detects overspending based on average trend  

---

## 🎯 Example

Add expenses → Analyze → Predict next spending  

Output:  
Predicted next expense: 850  
Warning: Possible overspending ahead
